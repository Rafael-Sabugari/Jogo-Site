The Robot War

O The Robot War é um jogo multiplayer de tiro, em que os jogadores controlam robôs com o intuito de se enfrentarem, para deixar o jogo mais divertido foram colocadas plataformas, algumas com armadilhas para fazer com que o player tenha uma certa vantagem sob o outro, no entanto, para não deixar o jogo sem graça os jogadores conseguem sair dessas armadilhas com facilidade. O intuito do jogo é de puro entreterimento ao usuário, por isso para não fazer com que o jogo perdesse esse intuito os desenvolvedores demoraram cerca de oito dias para finalizar esse jogo.
